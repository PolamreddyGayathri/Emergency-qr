# Emergency QR System

A web-based emergency response system that allows users to quickly access critical health information and emergency contacts by scanning a QR code. 

## Features

*   **Public Profile View:** Anyone scanning the QR code can see the user's blood group, emergency contacts, medical alerts, and address.
*   **Emergency Mode:** A dedicated emergency mode allows bystanders to quickly call the primary contact or send an emergency SMS.
*   **Protected Documents:** Access to crucial medical documents.
*   **Admin Dashboard:** A secure login for admins/users to manage profile information, update emergency contacts, edit medical alerts, and upload documents.
*   **Firebase Integration:** Powered by Firebase Authentication and Cloud Firestore for real-time data syncing and secure storage.
*   **Responsive UI:** A premium, fully responsive user interface built with Bootstrap 5 and AOS animations.

## Tech Stack

*   HTML5, CSS3, JavaScript (ES6 Modules)
*   Bootstrap 5 for styling
*   Firebase (Firestore & Authentication)
*   AOS (Animate On Scroll) for micro-animations

## Setup & Deployment

1.  Initialize Firebase with your project credentials.
2.  Deploy to Firebase Hosting using `firebase deploy` or host on any static server.
